## 1. 미분 기본 공식

### 핵심 공식

| 함수 | 미분 | 코드 | 직관 |
|---|---|---|---|
| $x^n$ | $nx^{n-1}$ | `n * x**(n-1)` | 지수가 앞으로, 지수는 1 줄어듦 |
| $e^x$ | $e^x$ | `np.exp(x)` | 자기 자신 |
| $\ln x$ | $1/x$ | `1/x` | $e^x$의 역함수 |
| $\sin x$ | $\cos x$ | `np.cos(x)` | 위상 90도 앞당겨짐 |
| $\tanh x$ | $1 - \tanh^2 x$ | `1 - t**2` | micrograd tanh._backward |
| $\text{ReLU}(x)$ | $\mathbf{1}[x > 0]$ | `(x > 0) * grad` | 음수 구간: 그래디언트 0 |
| $\text{softmax}(z_i)$ | $s_i(1 - s_j)$ | CrossEntropy와 합산 | 출력 - one_hot(y) |

### ReLU가 특별한 이유

```
x > 0 : 기울기 1 → 그래디언트 그대로 통과
x ≤ 0 : 기울기 0 → 그래디언트 완전 차단 (dead neuron 문제)
```

micrograd 구현:
```python
def relu(self):
    out = Value(max(0, self.data), (self,), 'ReLU')
    def _backward():
        self.grad += (out.data > 0) * out.grad  # ← 이 한 줄
    out._backward = _backward
    return out
```

## 2. 연쇄법칙 (Chain Rule)

### 수식

$$\frac{\partial f}{\partial x} = \frac{\partial f}{\partial u} \cdot \frac{\partial u}{\partial x}$$

### 직관

함수가 겹겹이 쌓여 있을 때 — **바깥 미분 × 안쪽 미분**.
아무리 깊게 쌓여도 이 곱셈을 반복하면 된다.

### 예시: $f(x) = (3x + 1)^2$, $x = 2$

```
u = 3x + 1      →   ∂u/∂x = 3
f = u²          →   ∂f/∂u = 2u = 2(3x+1)

∂f/∂x = ∂f/∂u × ∂u/∂x = 2(3x+1) × 3 = 6(3x+1)

x=2 대입: 6(3×2+1) = 6×7 = 42
```

PyTorch 검증:
```python
x = torch.tensor(2.0, requires_grad=True)
f = (3*x + 1)**2
f.backward()
print(x.grad)   # 42.0 ✓
```

### 신경망에서의 형태

$$\frac{\partial L}{\partial W_1} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial h} \cdot \frac{\partial h}{\partial W_1}$$

역전파는 이 곱셈을 오른쪽 → 왼쪽으로 자동 적용하는 것.

## 3. 계산 그래프와 역전파

### 순전파 vs 역전파

```
순전파 (왼쪽 → 오른쪽):  x → [*3, +1] → u → [²] → f
                                   값을 계산

역전파 (오른쪽 → 왼쪽):  x ← [×3]    ← u ← [×2u] ← ∂f/∂f=1
                                   연쇄법칙 적용
```

### 핵심 노드별 역전파 규칙

#### 덧셈 노드: $c = a + b$

$$\frac{\partial c}{\partial a} = 1, \quad \frac{\partial c}{\partial b} = 1$$

**그래디언트를 그대로 통과시킨다.**

```python
def __add__(self, other):
    out = Value(self.data + other.data, (self, other), '+')
    def _backward():
        self.grad  += out.grad   # × 1
        other.grad += out.grad   # × 1
    out._backward = _backward
    return out
```

#### 곱셈 노드: $c = a \times b$

$$\frac{\partial c}{\partial a} = b, \quad \frac{\partial c}{\partial b} = a$$

**서로의 값을 바꿔서 곱한다.**

```python
def __mul__(self, other):
    out = Value(self.data * other.data, (self, other), '*')
    def _backward():
        self.grad  += other.data * out.grad  # ← 상대방 값!
        other.grad += self.data  * out.grad
    out._backward = _backward
    return out
```

#### 거듭제곱 노드: $\text{out} = x^n$

$$\frac{\partial\,\text{out}}{\partial x} = n x^{n-1}$$

**고전적인 미분 공식 그대로.**

```python
def __pow__(self, n):
    out = Value(self.data**n, (self,), f'**{n}')
    def _backward():
        self.grad += (n * self.data**(n-1)) * out.grad
    out._backward = _backward
    return out
```

## 4. 위상 정렬 (Topological Sort)

### 왜 필요한가

역전파는 **출력 노드부터 입력 노드 방향**으로 _backward()를 호출해야 한다.
그런데 그래프가 복잡하면 순서를 어떻게 결정하나?

→ **위상 정렬**: 자식 노드를 먼저, 자신은 나중에 추가.
그 결과를 뒤집으면 "출력 먼저" 순서가 된다.

### 코드

```python
def backward(root):
    topo, visited = [], set()

    def build_topo(node):
        if node not in visited:
            visited.add(node)
            for child in node._prev:   # 자식 먼저
                build_topo(child)
            topo.append(node)          # 자신은 마지막

    build_topo(root)
    root.grad = 1.0                    # ∂L/∂L = 1

    for node in reversed(topo):        # 뒤집어서 순회
        node._backward()
```

### 직관

```
그래프:  x → u → f (순서)
topo:   [x, u, f]  (자식 먼저이므로 x가 제일 먼저)
역순:   [f, u, x]  ← 이 순서로 _backward() 호출
```

## 5. 행렬 버전 연쇄법칙

### 선형 레이어 $Y = XW$의 역전파

$$\frac{\partial L}{\partial X} = \frac{\partial L}{\partial Y} \cdot W^\top$$

$$\frac{\partial L}{\partial W} = X^\top \cdot \frac{\partial L}{\partial Y}$$

### Shape 추적

```
X: (N, D)   W: (D, M)   Y = X@W: (N, M)   dY: (N, M)

dX = dY @ W.T    →  (N,M) @ (M,D) = (N,D)  ← X와 shape 일치
dW = X.T @ dY    →  (D,N) @ (N,M) = (D,M)  ← W와 shape 일치
```

**규칙**: 그래디언트는 항상 파라미터와 **같은 shape**이어야 한다
(파라미터 업데이트에 직접 쓰이므로).

### NumPy MLP 역전파 코드

```python
# 3층 → 2층 방향
dZ3 = out - one_hot(y)           # softmax + CE 합산 미분
dZ3 /= N                         # 배치 평균

dW3 = A2.T @ dZ3                 # ∂L/∂W3 = A2ᵀ @ dZ3
db3 = dZ3.sum(axis=0)            # ∂L/∂b3  (브로드캐스팅 역전파)
dA2 = dZ3 @ W3.T                 # 아래 레이어로 흘려보낼 그래디언트

dZ2 = dA2 * (Z2 > 0)             # ReLU 역전파
dW2 = A1.T @ dZ2
...
```

## 6. Softmax + CrossEntropy 역전파

### 왜 함께 처리하는가

따로 미분하면 복잡하지만, 합산하면 깔끔해진다:

$$\frac{\partial L}{\partial z_i} = \hat{y}_i - y_i$$

($\hat{y}$: softmax 출력, $y$: one-hot 정답)

### 코드

```python
dZ = out.copy()            # softmax 출력 복사
dZ[range(N), labels] -= 1  # 정답 위치에서 1 빼기
dZ /= N                    # 배치 평균
```

직관: "정답 클래스 확률이 높을수록 그래디언트가 작아진다" — 맞게 예측했으면 덜 수정.
